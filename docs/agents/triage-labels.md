# Triage labels

The engineering skills use five canonical triage roles. The same strings are
used as GitHub labels in this repository.

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | メンテナーによる内容の確認・分類が必要 |
| `needs-info` | `needs-info` | 対応に必要な追加情報を待っている |
| `ready-for-agent` | `ready-for-agent` | エージェントが自律的に実装できる状態 |
| `ready-for-human` | `ready-for-human` | 人間による判断または実装が必要 |
| `wontfix` | `wontfix` | 対応しないことが決定済み |

When a skill refers to a canonical triage role, apply the corresponding GitHub
label from this table.
